Paragraph and Text Tags

This is a paragraph. Use <p> tags to wrap blocks of text. Extra spaces and line breaks inside HTML code are IGNORED by the browser. The browser treats all whitespace as a single space.

First line of text.
Second line — we used <br> to break to a new line.
Third line here.
Note: <br> only moves to the next line — it does NOT add extra vertical space like a new <p> does.

Text Formatting Tags

This text is in italics using the <em> tag.

This text is bold using the <strong> tag.

You can also combine both for bold italic text.

HTML is the standard language for creating web pages.

HTML Entities (Special Characters)

Use HTML entities to display reserved or special characters:

< = Less-than sign (&lt;)

> = Greater-than sign (&gt;)

& = Ampersand (&amp;)

© = Copyright symbol (&copy;)

  = Non-breaking space (&nbsp;) — used to add manual spaces in text

    This paragraph has 4 non-breaking spaces at the start (like an indent).
