15. Semantic & Interactive Tags

FAQ — Expand/Collapse with <details> and <summary>

What is the difference between <section> and <div>?

<section> is a semantic tag — it tells the browser and search engines that this block is a distinct thematic section of the page, usually with its own heading.

<div> is non-semantic — it's just a generic container with no meaning. Use it only for layout/styling when no semantic tag fits.

What is the difference between <strong> and <b>?

Both make text bold visually. But <strong> also tells screen readers and search engines "this content is important." <b> is purely visual — no semantic meaning. Always prefer <strong>.

What is the difference between id and class?

id = unique identifier, ONE element per page. Like an Aadhaar number.
class = reusable label, MANY elements can share it. Like a college batch.
